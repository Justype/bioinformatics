# Cheat Sheet

R Studio

| short cut        | Usage      |
| :--------------- | :--------- |
| `CTRL + ALT + I` | Code Chunk |

Rmd Chunk Options

| Option            | Usage            |
| :---------------- | :--------------- |
| `echo=FALSE`      | Hide source code |
| `results='hide'`  | Hide text output |
| `message=FALSE`   | Hide messages    |
| `warning=FALSE`   | Hide warnings    |
| `fig.show='hide'` | Hide plots       |
