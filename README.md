# Basic Terminal Shortcuts

| Shortcut | Usage |
| -- | -- |
| `Ctrl + C` | Stop |
| `Ctrl + L` | Clear |
| `Ctrl + U` | Erase Before Cursor |
| `Ctrl + K` | Erase After Cursor |
| `Ctrl + A` | Move Cursor to Start |
| `Ctrl + E` | Move Cursor to the End |
